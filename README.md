# DentistAI Scale - L'IA au service de l'art dentaire

Une agence de pointe aidant les cabinets dentaires à croître grâce à des systèmes d'IA éprouvés. Design minimaliste style Vercel avec animations fluides.

## Description

Cette application web présente les services de DentistAI Scale, une plateforme utilisant l'intelligence artificielle pour optimiser la croissance des cabinets dentaires. Elle garantit l'acquisition de 20 nouveaux patients en 30 jours ou le service est gratuit.

Fonctionnalités principales :
- Formulaire d'audit ROI pour analyser le potentiel de croissance
- Interface moderne avec animations et effets visuels
- Intégration d'un chatbot IA via ElevenLabs Convai
- Design responsive et optimisé pour les performances

## Technologies utilisées

- **React 19** avec TypeScript
- **Vite** pour le bundling et le développement
- **Tailwind CSS** pour le styling
- **ElevenLabs Convai** pour le chatbot IA

## Installation et exécution

### Prérequis

- Node.js (version 18 ou supérieure)

### Installation

1. Clonez le dépôt :
   ```bash
   git clone <url-du-repo>
   cd dentistai-scale---l-ia-au-service-de-l-art-dentaire
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Lancez l'application en mode développement :
   ```bash
   npm run dev
   ```

4. Ouvrez votre navigateur à l'adresse `http://localhost:3000`

### Build pour la production

```bash
npm run build
npm run preview
```

## Structure du projet

- `src/App.tsx` : Composant principal de l'application
- `src/components/` : Composants React réutilisables
- `src/index.tsx` : Point d'entrée de l'application
- `public/` : Assets statiques
- `vite.config.ts` : Configuration Vite
- `tsconfig.json` : Configuration TypeScript
